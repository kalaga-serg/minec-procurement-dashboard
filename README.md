# Minec procurement dashboard

Управленческий дашборд участия в закупке Минэкономразвития России на сопровождение официального сайта.

Включает:
- статус закупочного процесса;
- план/факт прошлого контракта;
- интерактивный сценарий цены;
- выводы по техническому заданию;
- матрицу рисков и действий.

Содержит реальные финансовые показатели. Репозиторий публичный.
